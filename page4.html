안내사항



  • 나의 기프트카드 관리 및 선물내역 확인은 [마이페이지>기프트카드] 에서 확인 가능합니다.
  • 기프트카드의 유효기간은 5년이며, 유효기간 만료 전 3회에 걸쳐 이메일로 안내드립니다.
  • 구매 금액은 1회 최소 1만원 이상, 최대 50만원 이하로 구매 가능합니다. (카드당 50만원 한도, 1인당 500만원 한도)
  • 기프트카드 바로구매 시 자동 기명등록 처리되어 온라인몰에서 바로 사용 가능합니다.
  • 기프트카드 바로구매 시 아이디당 관리 기준 초과 시 MMS로 발송됩니다.
  • 바로구매한 기프트카드를 매장에서 사용하려면, [마이페이지>기프트카드>나의 기프트카드] 화면에서 카드 터치 시 결제 바코드로 바로 사용 가능합니다.
  • 선물 받으신 기프트카드를 매장에서 사용하려면,
    문자로 받은 카드 이미지를 제시하시거나 온라인몰에 카드 등록 후 [마이페이지>기프트카드>나의 기프트카드] 화면에서 카드 터치 시 결제 바코드로 바로사용
    가능합니다.
  • 선물 받으신 기프트카드를 온라인몰에서 사용하려면, 카드 등록 후 바로 사용 가능합니다.
  • 최초 등록된 카드는 주 카드로 설정됩니다.
  • 구매취소는 금액 미사용 시에만 가능하며,
    구매일 기준 신용카드/계좌이체 결제수단은 14일 이내에 [마이페이지>기프트카드>나의 기프트카드] 메뉴 카드별 이용내역 상세 화면에서 취소 가능합니다.
  • 기프트카드 선물 거절 시 자동취소 됩니다.
  • 바로 구매한 경우 환불 신청은, 나이스정보통신 고객센터를 통해 환불 신청이 가능합니다.
    선물 받으신 경우 환불 신청은, 선물받은 MMS 내용 내 카드등록 정보를 확인하여 나의 기프트카드>환불신청 화면을 통해 신청 가능합니다.
  • 기프트카드 잔액 60%이상 사용 시 환불 가능합니다. 1만원권 이하일 경우, 잔액 80%이상 사용 시 환불 가능합니다.
  • 기프트카드의 잔액 이동은 불가합니다.
  • 기프트카드 환불 시 완료 전까지 철회가 불가합니다.
  • 기프트카드 환불완료 후 카드 사용이 불가합니다.
  • 잔액이 있는 기프트카드의 유효기간 만료 시 잔액 환불은 불가합니다.
  • 기프트카드의 잔액이 있는 경우 카드삭제가 불가합니다. CJ ONE 고객센터를 통해 삭제 요청 가능합니다.
  • 기프트카드 분실 시 사용 중지가 가능하며, 이 후 사용중지 해지를 통해 재사용이 가능합니다.
  • 기프트카드 오프라인 매장 사용시 일부 매장 제외
    사용불가 매장 확인하기 >
  • 온/오프라인 기프트카드 결제 및 반품 시스템 오류 발생 시, 데이터 내부 조정으로 인한 거래 내역이 일부 상이한 현상이 발생할 수 있습니다.
    (단, 고객님의 실질 거래 내용은 변함이 없습니다.)
